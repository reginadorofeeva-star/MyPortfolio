# Changelog

Все заметные изменения в этом проекте будут документированы в этом файле.

## [1.0.0] - 2025-06-04

### Добавлено
- ✨ Полная конвертация HTML портфолио в React приложение
- 🎨 9 React компонентов (Navigation, Hero, Projects, ProjectModal, Experience, Skills, Beyond, Footer, GradientBlobs)
- 📊 Структурированные данные о проектах в TypeScript
- 🎭 Модальные окна с детальной информацией о проектах
- 🔗 Плавная прокрутка по якорям
- ⌨️ Поддержка клавиши Escape для закрытия модальных окон
- 🌐 Ссылки на Behance для проектов
- 📱 Полностью адаптивный дизайн
- ⚡ Анимации fadeUp, pulse, drift
- 🎨 Темный дизайн с градиентными бликами
- 🔤 Импорт Google Fonts (Geist, Geist Mono, Instrument Serif)
- 📦 Константы для ссылок и навигации
- 🎯 TypeScript типы для всех данных
- 📖 README с полной документацией
- 🖼️ Lazy loading для изображений
- 🎨 SVG иконки в отдельных компонентах

### Оптимизировано
- ⚡ Производительность рендеринга компонентов
- 🎨 CSS переменные для легкой кастомизации
- 📦 Разделение кода на модули
- 🔧 Улучшенная структура проекта

### Технологии
- React 18
- TypeScript
- Tailwind CSS v4
- Vite
- Google Fonts

## Структура версий

Формат версий: [MAJOR.MINOR.PATCH]
- **MAJOR** — несовместимые изменения API
- **MINOR** — новая функциональность с обратной совместимостью
- **PATCH** — исправления багов с обратной совместимостью
