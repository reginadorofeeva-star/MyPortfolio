## 🎯 Секции сайта

4. **Skills** — Профессиональные навыки и инструменты
5. **Beyond** — Личные увлечения и фотографии

## 🚀 Как редактировать

### Изменить данные о проектах
Отредактируйте `src/app/data/projects.ts`:
- `projectsCards` — карточки на главной странице
- `projectsData` — детальная информация в модальных окнах

### Изменить личную информацию
Компонент `src/app/components/Hero.tsx`:
- Имя, должность, описание
- Теги навыков
- Контактные ссылки

### Изменить опыт работы
Компонент `src/app/components/Experience.tsx`:
- Массив `timeline` с датами, компаниями и должностями

### Изменить навыки
Компонент `src/app/components/Skills.tsx`:
- Массив `skills` — список навыков
- Массив `tools` — инструменты и технологии

### Изменить фотографии
Компонент `src/app/components/Beyond.tsx`:
- Массив `photos` с URL изображений

## 🎨 Цветовая схема

Темный минималистичный дизайн с акцентами:

- **Фон**: `#0a0a0a`, `#111111`, `#161616`
- **Акцент**: `#7c6af7` (фиолетовый)
- **Успех**: `#3dd68c` (зеленый)
- **Текст**: `#ededed` с различной прозрачностью

Все переменные находятся в `src/styles/theme.css`.

## ⌨️ Горячие клавиши

- **Esc** — закрыть модальное окно
- **Click на карточку проекта** — открыть детали
- **Click на ссылку навигации** — плавная прокрутка
