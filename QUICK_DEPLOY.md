# 🚀 Быстрое развертывание - 3 шага

## Вариант 1: Timeweb (РЕКОМЕНДУЕТСЯ для РФ) 🇷🇺

### Шаг 1: Соберите проект
```bash
npm run build
```
Это создаст папку `dist/` с готовым сайтом.

### Шаг 2: Зарегистрируйтесь и создайте хостинг
1. Перейдите на [timeweb.cloud](https://timeweb.cloud)
2. Зарегистрируйтесь
3. Создайте новый сайт (Виртуальный хостинг)
4. Купите или привяжите домен

### Шаг 3: Загрузите файлы
1. Подключитесь по SFTP (FileZilla, WinSCP)
   - Хост: ваш-сервер.timeweb.cloud
   - Порт: 22
   - Логин и пароль из панели Timeweb
2. Загрузите ВСЕ файлы из папки `dist/` в корень сайта
3. Готово! Сайт доступен по вашему домену ✅

---

## Вариант 2: Netlify (нужен VPN для первичной настройки)

### Шаг 1: Подготовка
```bash
npm run build
```

### Шаг 2: Drag & Drop
1. Откройте [app.netlify.com/drop](https://app.netlify.com/drop)
2. Перетащите папку `dist/` в окно браузера
3. Получите бесплатный домен вида `your-site.netlify.app`

---

## Вариант 3: Vercel

### Один шаг:
```bash
npm install -g vercel
vercel --prod
```

Следуйте инструкциям в терминале.

---

## После развертывания проверьте:

✅ Сайт открывается  
✅ Изображения загружаются  
✅ Модальные окна работают  
✅ Навигация функционирует  

## Обновление сайта

Когда нужно обновить сайт:
1. Внесите изменения в код
2. Запустите `npm run build`
3. Загрузите новый `dist/` на хостинг
4. Готово!
